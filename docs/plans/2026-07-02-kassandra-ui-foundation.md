# Kassandra UI — Design System + Landing Page (slice 1) — Design + Plan

> **For Claude:** REQUIRED SUB-SKILL: subagent-driven-development (per-task implement + review).

**Goal:** Stand up the Kassandra web UI as a Vite + React + TypeScript + Tailwind v4 SPA in a new `app/` package, adopting the **Delphi "warm parchment editorial with ember sparks"** visual language (see `docs/design/delphi-style-guide.md` — committed alongside this plan). Slice 1 = **(U1)** the design-system foundation (tokens, fonts, reusable component primitives) + **(U2)** a Kassandra landing page composed from those primitives. The functional dApp (wallet connect + protocol interactions via `@kassandra/sdk`) is the NEXT milestone — but wire the plumbing (pnpm workspace linking `app/` → `@kassandra/sdk`, wallet-adapter deps present) so it's ready.

## Decisions (locked with the user)
1. **Stack:** Vite SPA + React + TypeScript + Tailwind v4 + `@solana/wallet-adapter` (present, not yet used in slice 1) + `@kassandra/sdk` via the pnpm **workspace protocol**. Client-side dApp, no SSR.
2. **First slice:** design-system foundation + landing page (the Delphi guide is landing-oriented). Functional app = next milestone.
3. **Location:** a new top-level `app/` package; add a root `pnpm-workspace.yaml` (`packages: [sdk, app]`) so `app/` consumes `@kassandra/sdk` (`"@kassandra/sdk": "workspace:*"`).

## Design language — source of truth + adaptation
- **The Delphi style guide** (committed at `docs/design/delphi-style-guide.md`) is the visual spec. **Use the clean CSS custom-properties block at the bottom of the guide as the authoritative token source** — the prose tables have OCR artifacts (e.g. `Bronze #7f6…` truncated → the real value is `--color-bronze: #7f6e60`; ignore the nonsensical `--radius-full-2: 582px`). Tokens: parchment `#fdf6ee` canvas, soft-cream `#f0e6dc` nav, pure-card `#ffffff`, sepia `#2b180a` headings, bronze `#7f6e60` body, pebble `#d9cfc3` hairlines, chestnut `#3e2407` (the ONLY button fill) with a peach `#fed0b3` bloom-shadow, ember `#f65726` / saffron `#ff5c00` accents (sparingly).
- **Fonts (Google Fonts / @fontsource):** Martina Plantijn Light is proprietary → substitute **Cormorant Garamond** (weight 300 for whisper-light display; the guide's first-choice substitute alongside Lora) for display/headings, **Inter** (400/500) for body/UI, **Roboto Mono** (400) for the sparing code-accent (trigger conditions). Load locally (@fontsource) so the build is self-contained + offline.
- **Adapt Delphi's CONTENT to Kassandra** (a decentralized, AI-assisted OPTIMISTIC ORACLE on Solana) while keeping the visual language EXACTLY. Kassandra's value prop to weave in: propose-an-answer + challenge-window (optimistic resolution); AI-assisted verdicts (an open-source runner reruns a pinned model over the agreed facts, hashes-committed); economic security (proposer/fact/vote bonds, staker settlement, slashing, dead-end burns); futarchy governance (MetaDAO) for parameters/treasury; challenge markets. Tagline direction (pick/refine): e.g. "Truth, settled." / "An optimistic oracle with a mind." — editorial, not hypey.
- **Non-negotiable design rules (from the guide's Do/Don't):** parchment canvas everywhere (pure-card only for lifted cards); chestnut is the ONLY button fill (never blue/green); flat surfaces + hairline pebble borders (NO heavy drop shadows — the ONLY shadow is the peach button bloom + the trust-portrait gradient); serif (Cormorant) ONLY for display ≥20px, Inter for all body <20px; never pure black on cream body (use sepia/charcoal-bark; black reserved for nav/icon strokes); radii vocabulary is ONLY {4, 8, 12, 16, 70}px; ≤2 text colors per block; ember/saffron only 1–2 punctuation moments per viewport.

## Tasks

### U1 — Scaffold + design-system foundation
- **Workspace + scaffold:** add root `pnpm-workspace.yaml` (`packages: [sdk, app]`). Scaffold `app/` = Vite + React + TS (`pnpm create vite app --template react-ts` or equivalent), add Tailwind v4 (`tailwindcss` + `@tailwindcss/vite`), `@kassandra/sdk` as a `workspace:*` dep (import-smoke it — e.g. `import { PROGRAM_ID } from "@kassandra/sdk"` renders; confirm the SDK is built or build it), the Solana wallet-adapter deps (`@solana/wallet-adapter-react` + `-base` + `-wallets` + `@solana/web3.js` — present for the next milestone; NOT wired into UI yet), fonts via `@fontsource` (cormorant-garamond, inter, roboto-mono). `pnpm --filter app dev` runs; `pnpm --filter app build` succeeds.
- **Tailwind v4 `@theme`:** put ALL Delphi tokens (colors, the type scale `--text-*`/`--leading-*`/`--tracking-*`, spacing, radii, fonts) into the app's CSS `@theme` block (from the guide's Tailwind v4 block, corrected). Set the global canvas: `body` = parchment, default text = charcoal-bark/bronze via Inter, `-webkit-font-smoothing`.
- **Component primitives** (`app/src/components/ui/` — typed React components, Delphi-faithful): 
  - `Button` variants: **PrimaryChestnut** (chestnut fill, white text, 12px radius, ~16px/10px padding, Inter 15/500, the peach `#fed0b3` bloom box-shadow — the signature), **GhostOutline** (transparent, 1px pebble/charcoal border, sepia text), **NavPill** (soft-cream fill, sepia text, 12px radius).
  - `Card` (pure-card surface, 12–16px radius, 20–24px padding, 1px pebble border, NO heavy shadow), `EyebrowTag` (saffron/pebble Inter ~12px label, optional hairline pill), `SectionHeader` (centered: eyebrow + two-line Cormorant-300 display headline [40/56px] + one Inter ~17px bronze paragraph, max-width ~640px), `AvatarBubble` (70px circle image + optional `VerifiedDot` cobalt-blue check overlay — the only blue on the page), `TriggerPreviewCard` (nested sub-card, 8–12px radius, "When" label in driftwood, a condition line with the variable in ember orange, a "+ Add Action" row in stone).
- **A `/styleguide` route/page** rendering every primitive + the token swatches + the type scale — a living gallery to review the design system against the guide (this is the reviewable artifact for U1).
- Typecheck (`pnpm --filter app typecheck` / `tsc --noEmit`) clean + `pnpm --filter app build` succeeds + lint if configured. Commit `feat(app): scaffold Vite+React+Tailwind v4 + Delphi design-system primitives`.

### U2 — Kassandra landing page
- Compose the U1 primitives into `app/src/pages/Landing.tsx` (+ sections), Kassandra content in the Delphi editorial style, max-width 1200px, section-gap 80px:
  - **Nav bar** (soft-cream): left links (e.g. Protocol, Docs, Governance), centered "Kassandra" wordmark (Cormorant), right actions (e.g. "Explore" + "Connect" as a NavPill — the Connect is a placeholder button for the next milestone, not wired).
  - **Hero — the signature constellation:** a centered two-line Cormorant-300 display headline (Kassandra tagline) + one bronze subhead + the PrimaryChestnut CTA (+ GhostOutline secondary), surrounded by 6–10 scattered floating white "question cards" — ADAPT Delphi's testimonial cards to Kassandra: each shows a live-looking ORACLE QUESTION + a mini verdict/confidence or a proposer/challenger line (avatar + short quote), an ember highlight on the punchy word. This replaces a hero illustration (per the guide).
  - **"How it works" / "Why Kassandra"** section(s): centered SectionHeader + a 2-column feature grid or 3-up card row using `Card` — the optimistic lifecycle (propose → challenge window → AI reruns the pinned model over the facts → settle) with a `TriggerPreviewCard` sub-card rendering a Kassandra mechanic (e.g. `When challenged → AI verdict → settle`, the condition/variable in ember orange, monospace where apt).
  - **Trust / credibility** section: the Centered Portrait Panel pattern (16px radius, warm orange-red ambient, name/role overlay, soft bottom fade into parchment) — adapt to Kassandra (e.g. the open-source AI resolver / a "verified resolver" / an economic-security stat centerpiece) flanked by two stacked feature cards (economic security, futarchy governance).
  - **Footer** (soft-cream, hairline top border): wordmark, minimal link columns, a low-key line.
- Responsive (graceful stack on mobile; the constellation collapses to a simple stack/grid). Accessible (semantic landmarks, alt text, focus-visible rings in an on-brand tone, sufficient contrast — bronze-on-parchment passes AA for the sizes used; verify the ember/saffron usages are punctuation not body). No console errors.
- `pnpm --filter app typecheck` clean + `pnpm --filter app build` succeeds. Take/describe a screenshot or at least confirm the dev server renders the page. Update `app/README.md` (run/build, the Delphi design system, what's slice-1 vs next-milestone). Commit `feat(app): Kassandra landing page in the Delphi editorial style`.
